# 即刻动态

> 5 条候选,选适合语境的用

**A · 简洁版**
把安卓手机变成局域网云盘。扫码就能从电脑浏览器访问手机里的文件,不用装客户端,不用账号,文件不出门。MIT 开源。https://github.com/huangchengqian/HcqDrive

**B · 场景版**
"想从手机拽照片到电脑但微信压画质 / 数据线不在 / Airdroid 慢"——装个 HcqDrive,同 WiFi 浏览器扫码就行,文件不出局域网。

**C · 反向版**
"做云盘但故意不做云同步、跨网络、账号体系"——HcqDrive 的设计哲学是减法。只做一件事:WiFi 内手机↔设备的最快文件互传。

**D · 类比版**
像 AirDrop,但能传任意大小;像 Nextcloud,但服务器就是你的手机;像 LocalSend,但能持续浏览手机文件而不是一次性互传。

**E · 数字版**
M1 完结。27 个 Kotlin 源文件 + 63 个 Web 源文件 + 19 个 REST 端点 + 1700 行 API 契约。装一个 APK,扫码就能用。MIT 开源,在线 demo 可玩。
